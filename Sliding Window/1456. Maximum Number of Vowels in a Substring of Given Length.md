## Intuition

A basic sliding window solution.

## Solution

```java
class Solution {
    public int maxVowels(String s, int k) {
        int max = 0;
        int n = s.length();
        int vCount = 0;

        for (int i = 0, j = 0; i < n; i++) {
            char c1 = s.charAt(i);
            if (isVowel(c1))
                vCount++;

            if (i >= k) {
                char c2 = s.charAt(j);
                if (isVowel(c2))
                    vCount--;
                j++;
            }

            max = Math.max(max, vCount);
        }

        return max;
    }

    private boolean isVowel(char c) {
        return c == 'a' || c == 'e' || c == 'i' || c == 'o' || c == 'u';
    }
}
```
