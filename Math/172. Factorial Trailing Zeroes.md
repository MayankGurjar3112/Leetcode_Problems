## Problem Details
Title: "172. Factorial Trailing Zeroes"
Difficulty: "Medium"
Primary Topic: "Math"
tags: ["Math"]
---

## Intuition

Counting the highest n in 5^n in the given input.

---

## Solution

```java
class Solution {
    public int trailingZeroes(int n) {
        int result = 0;
        while (n / 5 > 0) {
            n /= 5;
            result += n;
        }
        return result;
    }
}
```
