---
## Problem Details
Title: "1657. Determine if Two Strings Are Close"
Difficulty: "Medium"
Primary Topic: "Hash Table"
Tags: ["Hash Table", "String", "Sorting", "Counting"]
Time Complexity: O(n)
Space Complexity: O(1)
---

## Intuition

Used 2 maps to keep track of the character and their freqencies and later on use it to check if the frequencies match.

---

## Solution

```java
class Solution {
    public boolean closeStrings(String word1, String word2) {
        int m = word1.length();
        int n = word2.length();
    
        if (m != n) return false;

        int[] map1 = new int[26];
        int[] map2 = new int[26];

        for (int i = 0; i < n; i++) {
            int j = word1.charAt(i) - 'a';
            map1[j]++;
        }
        for (int i = 0; i < n; i++) {
            int j = word2.charAt(i) - 'a';
            map2[j]++;
        }

        for (int i = 0; i < n; i++) {
            int j = word2.charAt(i) - 'a';
            if (map1[j] < 1) return false;
        }
        for (int i = 0; i < n; i++) {
            int j = word1.charAt(i) - 'a';
            if (map2[j] < 1) return false;
        }

        Arrays.sort(map1);
        Arrays.sort(map2);

        for (int i = 0; i < 26; i++) {
            if (map1[i] != map2[i]) return false;
        }

        return true;
    }
}
```
