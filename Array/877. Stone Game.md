## Intuition

Though the question's topics do not suggest but a Two Pointer approach is the best solution for the problem, where u always keep a pointer at both ends and select the biggest value for Alice and smallest value for Bob and move the pointers.

---

## Solution

```java
class Solution {
    public boolean stoneGame(int[] piles) {
        boolean turn = true;
        int a = 0, b = 0;
        int i = 0, j = piles.length - 1;
        while (i <= j) {
            if (turn) {
                if (piles[i] > piles[j]) {
                    a += piles[i++];
                } else {
                    a += piles[j--];
                }
            } else {
                if (piles[i] > piles[j]) {
                    a += piles[j--];
                } else {
                    a += piles[i++];
                }
            }
        }
        return a > b;
    }
}
```
