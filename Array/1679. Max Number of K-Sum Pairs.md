## Problem Details
Title: "1679. Max Number of K-Sum Pairs"
Difficulty: "Medium"
Primary Topic: "Array"
Tags: ["Array", "Hash Table", "Two Pointers", "Sorting]
Time Complexity: O(n)
Space Complexity: O(n)
---

## Intuition

Similar to Two Sum solution but with a counter.

---

## Solution

```java
class Solution {
    public int maxOperations(int[] nums, int k) {
        int count = 0;
        Map<Integer, Integer> map = new HashMap<>();

        for (int i : nums) {
            int diff = k - i;
            if (diff <= 0) continue;
            if (map.containsKey(i)) {
                int amt = map.get(i);
                count++;
                if (--amt == 0)
                    map.remove(i);
                else 
                    map.put(i, amt);
            } else {
                map.put(diff, map.getOrDefault(diff, 0) + 1);
            }
        }

        return count;
    }
}
```
