## Problem Details
Title: "53. Maximum Subarray"
Difficulty: "Medium"
Primary Topic: "Array"
Tags: ["Array", "Divide and Conquer", "Dynamic Programming"]
Time Complexity: O(n)
Space Complexity: O(1)
---

## Intuition

Solution using Kadane's algorithm.
---

## Solution

```java
class Solution {
    public int maxSubArray(int[] nums) {
        int sum = 0, max = -10001;

        for (int i = 0; i < nums.length; i++) {
            sum += nums[i];
            max = sum > max ? sum : max;
            if (sum < 0)
                sum = 0;
        }

        return max;
    }
}
```
