---
## Problem Details
Title: "377. Combination Sum IV"
Difficulty: "Medium"
Primary Topic: "Dynamic Programming"
Tags: ["Array", "Dynamic Programming"]
Time Complexity: O(n * target)
Space Complexity: O(target)
---

## Intuition

Standard DP solution, exploring all the values in `nums` for all values of target and keeping the minimum value in all calls

---

## Solution

```java
class Solution {
    int[] memo;
    String s;
    int n;

    public int numDecodings(String s) {
        if (s.charAt(0) == '0')
            return 0;
        this.s = s;
        n = s.length();
        memo = new int[n];

        return dp(0);
    }

    private int dp(int i) {
        if (i >= n || s.charAt(i) == '0')
            return 0;
        if (memo[i] != 0)
            return memo[i];
        if (i == n - 1)
            return memo[i] = 1;

        int keep = dp(i + 1);

        int curr = s.charAt(i) - '0';
        int next = i + 1 < n ? s.charAt(i + 1) - '0' : 100;

        int skip = 0;
        if (curr * 10 + next < 27) {
            if (i == n - 2)
                skip = 1;
            else
                skip = dp(i + 2);
        }

        return memo[i] = keep + skip;
    }
}
```
