---
## Problem Details
Title: "377. Combination Sum IV"
Difficulty: "Medium"
Primary Topic: "Dynamic Programming"
Tags: ["Array", "Dynamic Programming"]
Time Complexity: O(n * target)
Space Complexity: O(target)
---

## Intuition

Standard DP solution, exploring all the values in `nums` for all values of target and keeping the minimum value in all calls

---

## Solution

```java
class Solution {
    int[] memo;
    int n;
    int[] nums;
    
    public int combinationSum4(int[] nums, int target) {
        this.nums = nums;
        n = nums.length;
        memo = new int[target + 1];

        Arrays.fill(memo, -1);

        return dp(target);
    }

    private int dp(int t) {
        if (t < 0) return 0;
        if (memo[t] != -1) return memo[t];
        if (t == 0) return 1;
        int sum = 0;

        for (int i = 0; i < nums.length; i++) {
            sum += dp(t - nums[i]);
        }

        return memo[t] = sum;
    }
}
```
