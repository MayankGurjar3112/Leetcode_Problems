---
## Problem Details
Title: "416. Partition Equal Subset Sum"
Difficulty: "Medium"
Primary Topic: "Dynamic Programming"
Tags: ["Array", "Dynamic Programming"]
Time Complexity: O(n * target)
Space Complexity: O(n * target)
---

## Intuition

A Top Down recursive solution, isntead of using a boolean memo, I used a integer memo to store 3 states, -1 for not visited, 0 for visited, 1 if the answer is true.

---

## Solution

```java
class Solution {
    int[][] memo;
    int[] nums;
    int n;

    public boolean canPartition(int[] nums) {
        int sum = 0, half = 0;
        n = nums.length;
        this.nums = nums;

        for (int i : nums) sum += i;
        if (sum % 2 == 1) return false;
        half = sum / 2;

        memo = new int[n][half + 1];
        for (int[] i : memo) Arrays.fill(i, -1);

        return dp(0, half) == 1;
    }

    private int dp(int i, int t) {
        if (i >= n || t < 0) return 0;
        if (memo[i][t] != -1) return memo[i][t];
        if (t == 0) return 1;

        int skip = dp(i + 1, t);
        int take = dp(i + 1, t - nums[i]);

        return memo[i][t] = Math.max(skip, take);
    }
}
```
