## Problem Details
Title: "68. Text Justification"
Difficulty: "Medium"
Primary Topic: "String"
tags: ["Arrays", "String", "Simulation"]
---
## Intuition
The problem had various rules and format that needed to be followed and the final output needed to be a modified version of the input. So the problem seemed like a simulation problem. 
I looped through the input array and track the current size of the lines, and when the size of line exceeded the `maxWidth`, I processed the strings that fitted in that line with appropriate justifiation.
And added a few special cases for lines with single words or last lines.
---
## Solution
```java
class Solution {
    public List<String> fullJustify(String[] words, int maxWidth) {
        List<String> list = new ArrayList<>();
        StringBuilder sb = new StringBuilder();
        int start = 0, end = 0, size = 0, gap = 0;
        while (end <= words.length) {
            int len = maxWidth;
            if (end < words.length)
                len = words[end].length();
            if (size + gap + len <= maxWidth) {
                size += len;
                gap++;
                end++;
            } else {
                gap--;
                int space = maxWidth - size;
                int wordCount = end - start;
                int addSpace = space / (wordCount - 1 == 0 ? 1 : wordCount - 1);
                int remSpace = space % (wordCount - 1 == 0 ? 1 : wordCount - 1);

                for (int i = start; i < end; i++) {
                    sb.append(words[i]);
                    if (end == words.length) {
                        sb.append(" ");
                    } else if (i < end - 1 || wordCount == 1) {
                        sb.append(" ".repeat(addSpace));
                        if (remSpace > 0)
                            sb.append(" ");
                        remSpace--;
                    }
                }

                if (end == words.length && maxWidth - sb.length() > 0) {
                    sb.append(" ".repeat(maxWidth - sb.length()));
                }
                sb.setLength(maxWidth);
                list.add(sb.toString());
                sb.setLength(0);
                size = 0;
                gap = 0;

                start = end;
            }
        }
        return list;
    }
}
```
