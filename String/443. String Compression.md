---
## Problem Details
Title: "443. String Compression"
Difficulty: "Medium"
Primary Topic: "Two Pointer"
tags: ["Two Pointers", "String"]
Time Complexity: O(n)
Space Complexity: O(1)
---

## Intuition

Simple Two Pointer approach, where one pointer tracks the current character and its count and the other updates the array when the counting of that said character is complete.

---

## Solution

```java
class Solution {
    public int compress(char[] chars) {
        int i = 0, j = 0;
        int n = chars.length;
        char cChar = (char) '\s';
        int cCount = 1;

        while (i <= n && j < n) {
            char curr = (char) '\s';
            if (i != n)
                curr = chars[i];
            if (cChar == curr) {
                cCount++;
            } else {
                if (cChar != (char) '\s') {
                    int t = 1;
                    while (cCount / t >= 10) {
                        t *= 10;
                    }
                    chars[j] = cChar;
                    if (cCount > 1)
                        while (t > 0) {
                            if (j + 1 < n)
                                chars[++j] = (char) (cCount / t + '0');
                            cCount %= t;
                            t /= 10;
                        }
                    j++;
                }
                cCount = 1;
                cChar = curr;
            }
            i++;
        }

        return j;
    }
}
```
