## Problem Details
Title: "2390. Removing Stars From a String"
Difficulty: "Stack"
Tags: ["String", "Stack", "Simulation"]
Time Complexity: O(n)
Space Complexity: O(n)
---

## Intuition

Used StringBuilder as a stack and got a little bit easier and better solution.

---

## Solution

```java
class Solution {
    public String removeStars(String s) {
        char[] chars = s.toCharArray();
        StringBuilder sb = new StringBuilder();

        for (char c : chars) {
            if (c == '*') {
                sb.setLength(sb.length() - 1);
            } else {
                sb.append(c);
            }
        }

        return sb.toString();
    }
}
```
